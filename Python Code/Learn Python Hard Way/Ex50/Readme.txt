This is Skeleton Folder for making new projects according to Book Learn Python ard Way

Steps to make this work

1. Replace NAME with the name you want to give to your projects

2. Make modules and store it in the NAME(to be replaced) Folder

3. Configure the setup.py file according to needs

4. Place the scipts you want to use later on after installing the distribution

5. when all is done run command in the root directory
    python setup.py sdist

6. open the dist folder and run the command
    pip install file_name.tar.gz
    file_name is Project Name with the vesion attached

7. The setup is installed correctly and use the scripts
